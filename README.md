# MUSIC RECOMMENDATION
NLP Based project of Music Recommendation. This is done using a dataset of top 10000 songs from Kaggle. I had preprocessed the columns and created a unified column named "Tags". Which is used to uniquely distinguish songs and also find the similarity between them, Just the same way how e-commerce sites work. This data was converted to Vectors using CountVectorizer and then to get the similarities between each song with each other song, I've used Cosine Similarity. 
And finally, the project is deployed and hosted as a streamlit application on Huggingface.
